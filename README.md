
## Deployment

Clone the project

```bash
  git clone https://github.com/Porgramming-Hero-web-course/l2b2-full-stack-a5-server-side-jubayer1418.git
```

Go to the project directory

```bash
  cd your-project-directory
```

Install dependencies

```bash
 npm install
```
Start the server

```bash
  npm run dev
```
